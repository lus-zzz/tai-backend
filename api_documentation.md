# 前端交互 API 文档

## 1. 设备管理 API

### 设备信息
- **获取所有设备信息**：返回所有设备的基本信息，包括设备 ID、IP、名称、端口等。
- **获取设备状态**：返回指定设备的在线状态和版本信息。
- **获取设备关联任务**：返回与指定设备关联的任务列表。
- **创建设备**：创建一个新的设备记录。
- **删除设备**：删除指定设备记录。
- **更新设备信息**：更新指定设备的基本信息。

### 数据中心相关
- **获取数据中心列表**：返回指定设备的数据中心列表。
- **获取数据存储列表**：返回指定数据中心的存储列表。
- **获取网络列表**：返回指定数据中心的网络列表。
- **获取集群列表**：返回指定数据中心的集群列表。
- **获取主机列表**：返回指定集群的主机列表。
- **获取虚拟机列表**：返回指定主机的虚拟机列表。
- **获取文件夹列表**：返回指定数据中心的文件夹列表。
- **获取文件夹虚拟机列表**：返回指定文件夹的虚拟机列表。
- **获取虚拟机详细信息**：返回指定虚拟机的详细信息。
- **检查虚拟机是否存在**：检查指定虚拟机是否存在。

## 2. 任务管理 API

### 任务信息
- **获取所有任务信息**：返回所有任务的基本信息，包括任务 ID、名称、设备列表等。
- **创建任务**：创建一个新的任务记录。
- **删除任务**：根据任务 ID 列表删除任务记录。
- **更新任务信息**：更新指定任务的基本信息。
- **导入任务**：从外部导入任务记录。
- **启动任务**：启动指定任务。
- **停止任务**：停止指定任务。

### 任务版本管理
- **获取任务版本列表**：返回指定任务的版本列表。
- **设置版本锁定状态**：设置指定版本的锁定状态。
- **删除任务版本**：删除指定任务的版本记录。

### 任务相关操作
- **检查虚拟机服务状态**：检查指定虚拟机的服务状态。
- **获取任务版本虚拟机列表**：返回指定任务版本的虚拟机列表。
- **检查目标文件夹**：检查目标文件夹是否存在。
- **获取磁盘分区信息**：返回磁盘的分区信息。
- **获取分区系统信息**：返回分区的系统信息。
- **下载分区系统文件**：下载指定分区的系统文件。

## 3. 还原任务 API

### 还原任务信息
- **获取所有还原任务信息**：返回所有还原任务的基本信息。
- **创建还原任务**：创建一个新的还原任务记录。
- **删除还原任务**：删除指定还原任务记录。
- **启动还原任务**：启动指定还原任务。
- **停止还原任务**：停止指定还原任务。

## 4. 其他 API

- **获取系统概览信息**：返回系统的概览信息。
- **记录操作日志**：记录用户操作日志。
- **导出调试日志**：导出系统的调试日志。

## 5. 版本管理 API

- **获取任务版本列表**：返回指定任务的版本列表。
- **设置版本锁定状态**：设置指定版本的锁定状态。
- **删除任务版本**：删除指定任务的版本记录。
